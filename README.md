# Potion Power BI Gallery

This repository contains a curated collection of Power BI dashboards and visual components,
focused on business analysis, data storytelling, and professional dashboard design.

## What's inside
- Commercial dashboards
- Financial analysis
- Marketing performance
- People Analytics
- Reusable visual components (KPIs, gauges, charts)

## Purpose
- Share practical Power BI examples
- Help analysts learn advanced modeling and DAX
- Showcase real-world dashboard patterns

## How to use
1. Download the PBIX file
2. Replace the sample dataset if needed
3. Explore measures, visuals and layout techniques
